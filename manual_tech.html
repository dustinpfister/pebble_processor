Pebblebar is pebbles plug-in collection

  • plug-in folders:

  • plug-in files:

    a plugin may have one or more of the following files in it.

    • conf.json

    • updater.js

    • responder.js

    • setup.js

    • fulfiller.js

      A fulfiller.js file is what is called when a pebble transaction is finished being processed for the given plug-in. It is used to preform actions that are to be completed in the event that a pebble transaction succeeds or fails.
